# Redontex PayU Latam – Módulo de pago para PrestaShop

Integración de PayU Latam para PrestaShop desarrollada por **Redontex**.  
Este módulo permite procesar pagos con PayU de forma segura, estable y totalmente integrada con el flujo estándar de pedidos de PrestaShop.

## Características
- Integración directa con la API de PayU Latam.
- Pagos con tarjeta y métodos locales según país.
- Validación de firma digital.
- Actualización automática del estado del pedido.
- Arquitectura limpia y mantenible.

## Compatibilidad
- PrestaShop **1.7.x - 8.x**
- PHP **7.2 – 8.1**
- Servidores Apache / Nginx

## Instalación
1. Descargar el módulo en formato `.zip`.
3. Subirlo desde **Módulos → Subir un módulo** en el backoffice de PrestaShop.
4. Configurar las credenciales de PayU:
   - API Key  
   - Merchant ID  
   - Account ID  
   - Seleccionar modo: pruebas / producción

## Entorno de pruebas
PayU ofrece un entorno sandbox para validar transacciones sin coste.  
Este módulo incluye soporte completo para sandbox.
